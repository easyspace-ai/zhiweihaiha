# PptxGenJS Studio QA

## 结构

- [ ] 有 `cover_hero` 首页与 `closing_cta` 末页
- [ ] 至少 1 次 `section_break`
- [ ] 至少 1 次 `stat_row` 或 `chart_sidebar`
- [ ] 无连续 3 页相同 `layout_preset`
- [ ] ≥80% 页使用 `layout_preset`（非手写 elements）

## 内容

- [ ] 无 lorem / 占位符 / 「待补充」
- [ ] footer 为短章节标签，非整份报告名
- [ ] 数字来自用户材料

## 视觉

- [ ] 存在深色页与浅色页交替
- [ ] 单页 bullet ≤ 6 条，每条 ≤ 90 字
- [ ] chart 数据与文案一致

失败时：回到阶段 2 只改 slots 文案，不换 preset 序列（除非叙事问题）。
