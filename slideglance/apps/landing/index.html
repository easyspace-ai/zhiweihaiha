Rust + WebAssembly · MIT

Open any .pptx. Author your own from .sgx. One Rust + WebAssembly engine, runs locally.

SlideGlance pairs two tools on one Rust + WebAssembly engine: a PPTX viewer that opens decks without uploading them, and a declarative XML builder that compiles .sgx source into deterministic .pptx — with a 30-rule lint pass that catches overflow and visual drift before the file leaves your hands.

Two tools, one engine

The viewer and the builder are independent products — same Rust + WebAssembly core underneath, distinct surface above. Use either, use both.

View

Render any .pptx

Drop a PowerPoint deck onto the page, the Chrome extension, the desktop app, or any host that embeds the React shell. A Rust renderer parses the OOXML, resolves theme + master inheritance, shapes text, and paints SVG or PNG — entirely in the WebAssembly sandbox.

  • Selectable <text> SVG (copy + search + a11y).
  • Theme fonts, master fills, tables, charts, gradients, EMF.
  • Deterministic — same input + pinned fonts → byte-identical SVG.
Viewer details →
Build

Author .sgx declaratively

Decks-as-code: a tiny XML grammar with Flexbox-like layout primitives, named templates, styles, masters, and <Foreach> over data. Compile to PowerPoint with autoFit, pixel-aware text measurement, and a 30-rule lint pass.

  • Versioned XSD + JSON Schema for IDE autocomplete.
  • Lint rules for overflow, contrast, font drift, a11y, perf.
  • Live-preview VS Code extension with click-to-source.

Install — every surface

Same Rust + WebAssembly core, six surfaces. Pick the one that fits where the deck lives.

Chrome Web Store

Right-click any .pptx link in Chrome → "Open with SlideGlance". Same React viewer the playground uses, framed as a new-tab page.

apps/chrome-extension

VS Code Marketplace

SlideGlance PPTX Viewer extension — schema-aware editing, live PPTX preview, one-click export. Click any shape in the preview to jump to its source line.

apps/vscode-extension

Desktop viewer

Native macOS / Windows / Linux app built with Tauri. Open .pptx from the OS file manager without a browser hop. Installers are attached to each GitHub release.

apps/desktop-viewer

npm packages

@slideglance/builder (XML → PPTX), /viewer (React shell), /core + /measure (WASM bindings to the Rust renderer + text measurer).

4 packages, MIT

crates.io

Native CLI slideglance convert deck.pptx --output out/ plus the underlying renderer / parser / font / png / model crates. Deterministic, no Node runtime needed.

9 crates, Rust 2021

Web playground

No install — drag any .pptx onto the page. The whole renderer ships as a single WebAssembly bundle and runs in your tab, including the React viewer chrome.

apps/web-playground

The web playground is a drag-and-drop SPA — drop any .pptx on the page and it renders entirely in your tab.
