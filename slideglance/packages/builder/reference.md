### `<When>`

### `<Otherwise>`

Content model: `any`. Allowed parents: `<Choose>`.

### `<Foreach>`

Repeats its body once per element of `items`. `items` is parsed as a JSON array after placeholder substitution, so both inline literals (items='[{...}]') and references to a parent attribute (items="{milestones}") are accepted. Each iteration binds `as`, plus optional indexAs/firstAs/lastAs vars, into the scope.

Content model: `any`. 

**Attributes**

| Name | Type | Required | Description |
|---|---|---|---|
| `as` | string | yes | Name bound to the current item inside the body. |
| `firstAs` | string |  | Optional name bound to true on the first iteration, false otherwise. |
| `indexAs` | string |  | Optional name bound to the 0-based index of the current iteration. |
| `items` | string | yes | JSON array literal, or a {ref} that resolves to one after substitution. |
| `lastAs` | string |  | Optional name bound to true on the last iteration, false otherwise. |
