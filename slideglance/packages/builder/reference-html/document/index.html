<Document>

Document settings — slide size, default master, default text style. One <Document> per <SlideGlance>.

Attributes

NameTypeRequiredDescription
boldbooleanDefault bold.
colorcolorDefault text color.
defaultMasterstringMaster name used by slides that omit `master`.
fontFamilystringDefault text fontFamily.
fontSizenumberDefault text fontSize.
hnumberAlias of `height`.
heightnumberCustom slide height (px). Pair with `width`. Mutually exclusive with `size`.
highlightcolorDefault highlight colour.
italicbooleanDefault italic.
lineHeightnumberDefault lineHeight multiplier.
sizestringPreset slide size. Mutually exclusive with width/height/w/h.
strikebooleanDefault strikethrough.
underlineunderlineDefault underline (boolean or object).
wlengthAlias of `width`.
widthlengthCustom slide width (px). Pair with `height`. Mutually exclusive with `size`.

Allowed children

This element accepts no child elements.
