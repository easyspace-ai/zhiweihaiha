# npm packages

This directory contains the published npm packages. Each one is a
thin wrapper over the Rust workspace, exposing a JS-native API.

For the navigated list with per-package doc links, see
[`docs/en/packages.md`](../docs/en/packages.md).

Each package ships its own documentation triple under
`<package>/docs/en/{index,reference,guides}.md`.
